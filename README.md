# Dr. Rohan Shetty — Surgical Oncologist & Robotic Surgeon

Official website of Dr. Rohan Shetty, MBBS, MS (Gen. Surg), MRCS (Edin.), FRCS (Glasgow), FACS, FSSO.

Surgical oncologist and robotic surgeon based in Mangalore, India, with 16+ years of experience and over 5000 major cancer surgeries.

## Tech

- Single-page static site (HTML + Tailwind CSS + vanilla JS)
- Spline 3D animated hero background (desktop), gradient fallback on mobile
- Fully responsive with mobile hamburger navigation
- SEO optimised: meta tags, Open Graph, Twitter cards, JSON-LD Physician schema, sitemap.xml, robots.txt
